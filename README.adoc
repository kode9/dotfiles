|kbd:[C-b] |kbd:[Ã©] |split-window -v |Split window vertically

|kbd:[C-b] |kbd:[K] |kill-window |Kill window

|kbd:[C-b] |kbd:[:] |copy-mode |Enter copy mode

|kbd:[C-b] |kbd:[!] |paste-buffer|Paste copy buffer into current pane
